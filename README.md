---NA SUA ÁREA DE TRABALHO/DESKTOP
1. Instalar VSCode (não deixe ele aberto depois de instalar)
2. Instalar Git
3. Instalar Node.js
4. Crie uma pasta aonde você vai guardar os arquivos do chatbot (Sugestão de nome: Chatbot)
5. Cadastrar e logar no github

---DENTRO DO VSCODE (agora sim pode abrir)

6. Conectar seu github dentro do VSCode
7. Abra a pasta que você criou no passo 4 (Chatbot)
8. Abrir terminal (cmd, não powershell!)
9. Use o comando: git clone https://github.com/4555503510-cell/lacerdinha/
    
---ÁREA DE TRABALHO

  Agora você tem os arquivos básicos na sua pasta!   
10. Copie eles (dentro da pasta /lacerdinha/) e cole na pasta que você criou (Chatbot)

---Terminal do VSCODE (cmd, sempre)

11. Use o comando:

npm install

14. Deve dar um erro, mas não se preocupe! Agora aperte Windows+R para abrir a barra de execução

15. Digite: %userprofile%\.cache

16. Você vai ser levado pra pasta ".cache", e nela vai encontrar a pasta "puppeteer".

17. Entre na pasta "puppeteer", e depois na pasta "chrome"

18. Aqui você vai encontrar duas pastas! Entre na SEGUNDA

19. Copie a pasta "chrome-win64" que você encontrou ali

20. Volte pra pasta anterior, e entre na PRIMEIRA

21. Cole a pasta "chrome-win64" aqui!

22. Se ele pedir pra substituir 3 arquivos, ótimo! Estamos quase prontos

--VSCode
21. Mais uma vez, digite no terminal:

npm install

22. Espere e dessa vez não deve ter erros

23. Agora é só rodar o bot com o comando:

node chatbot.js

24. Espere e você vai receber um QRCode, que deve escanear com seu celular à partir da tela de "Conectar Dispositivos" do WhatsApp

25. Depois que escanear, espere um pouco e fique de olho no terminal do VSCode: você vai receber um aviso falando "Chatbot is Ready!" pra saber que o bot está rodando

26. Teste o bot enviando "testecedup2026" pro número conectado!
